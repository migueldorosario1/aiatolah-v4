# Domínio e laboratório no ar — Aiatolah

O Aiatolah já tem domínio e uma página experimental no ar:

```text
https://aiatolah.com/
```

## Como interpretar isso

Este site **não deve ser tratado como referência obrigatória de design, conteúdo ou arquitetura final**.

Ele existe principalmente para mostrar que:

- o domínio já está ativo;
- a infraestrutura inicial já responde;
- há um laboratório de produção disponível;
- o projeto pode evoluir a partir de algo já publicado.

## O que ainda está em aberto

O projeto editorial, visual, técnico e arquitetural do Aiatolah continua em aberto.

O ChatGPT não deve assumir que a página atual define:

- layout final;
- identidade visual final;
- categorias definitivas;
- tom editorial definitivo;
- estrutura de publicação definitiva;
- arquitetura final dos agentes.

## Regra para o ChatGPT

Use o fato de que o domínio já está no ar apenas como dado operacional:

```text
Já existe um domínio ativo e um laboratório publicado em https://aiatolah.com/, mas o projeto final está em aberto. Não trate a página atual como referência obrigatória. Proponha livremente arquitetura, conteúdo, design, agentes e estratégia editorial para transformar o Aiatolah em um portal de inteligência artificial.
```

## Pedido ao ChatGPT

Depois de ler este arquivo, responda:

```text
Sabendo que o domínio Aiatolah já está no ar, mas que o projeto final está totalmente em aberto, qual seria a melhor arquitetura editorial e técnica para criar um portal de inteligência artificial bilíngue, automatizado e com perspectiva de Sul Global?
```
