The 'Apollo-11' repository on GitHub, maintained by Chris L. Garry, provides the original source code of the Apollo Guidance Computer (AGC) for the command module (Comanche055) and lunar module (Luminary099). The material was digitized by the Virtual AGC and the MIT Museum.

According to GitHub, the repository aims to preserve the original Apollo 11 code. Contributions are welcome to correct discrepancies between the transcriptions and the original scans of the Luminary 099 and Comanche 055 files.

The command module code, called Colossus 2A, was assembled on April 1, 1969, revision 055. The lunar module code, Luminary 1A, was assembled on July 14, 1969, revision 001. Both use the yaYUL assembler.

The code is in the public domain. The digitization was done by Paul Fjeld, organized by Deborah Douglas of the MIT Museum. The program was developed under contract NAS 9-4065 with NASA by the MIT Instrumentation Laboratory.

Margaret H. Hamilton, lead programmer of Colossus, signed the submission on March 28, 1969. Approvals came from Daniel J. Lickly, Fred H. Martin, Norman E. Sears, Richard H. Battin, David G. Hoag, and Ralph R. Ragan, all on the same date.

The repository includes instructions for compiling the original code via Virtual AGC. Those interested in computer history and space exploration can access the code and contribute improvements.
